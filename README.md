# Yume2kki_IMEDictionary
ゆめ2っきの日本語マップ名を変換するための辞書です。いまのところほぼ個人制作です。
MS-IME用とGoogle日本語入力用があります。バージョン末尾「M」がMS-IME用、「G」がGoogle日本語入力用です。
基本的に最新バージョンを使ってください。更新は不定期ですので最新マップは対応していない場合があります。
最新バージョンのzipの中に大バージョンとの差分も同梱しております。



これわからない人稀に居るんで書いときます。**Release欄に公開しております。**
MS-IME用はBOM付きUTF-16じゃないとダメとかあるんで、テキストだけコードとして普通に公開してもそのまま使えないんですね。なのでzip化したのちRelease機能を使って公開しております。
私にはこれより賢いやり方がわからなかった。そもそもGitHubでの公開ほぼ初見だし。許してね。

｜☆　マップじしょ　☆｜
よみを入力するとマップ名が変換できるものです。地味に変換が大変だったあのマップを一発で。

homeやInterludeなど英字マップ名に関しては、ローマ字入力を切らずに入力した場合と、英語を読み上げた際の正しい読み両方を登録してあります。
例:単語「home」 読み「ほめ」「ほーむ」

その他特殊なものについて
単語「63317664」読み「６３３１７６６４」
単語「94nd0r4」読み「ぱんどら」「９４んｄ０ｒ４」※94nd0r4はPAndOrAのリート表記
単語「GrayMemory」読み「ｇらｙめもｒｙ」「ぐれいめもりー」「ぐれいめもり」
単語「(＾□＾))」読み募集中！「えがお」「おわた」「かおもじ」「はっきょう」「(＾しかく＾)」
単語「lim Δt→0」読み「りみっとでるたてぃーぜろへ」「りみっとでるたｔ０へ」「ぃみｔでるたｔみぎ０」

ちゅういでも書きましたが読みに不備があったりした場合は私にお申し付けください。取り入れるかもしれません。



｜☆　そのたじしょ　☆｜
エフェクト名、ミニゲーム名、オブジェ名、オンライン関連などが中心です。
NPC名は正直網羅できる気がしない。



｜☆　パッチ　☆｜
ファイル名に書いてある辞書の、大バージョンとの差分ファイルです。Ver何から何へなのかはファイル名にそのまま記載してます。
各バージョンに収録しているメインの辞書は全て完全版なので、前のバージョンから使ってくれてた人向け。
あとMS-IMEは辞書管理が結構だるいので「とりあえずぶち込んどけばいいやつ」を作りたかった。
そういう人には完全版が邪魔くさいと思いますが、GitHubリリース増やしすぎるのも見にくいと思いこういった形にさせていただきました。
いらんファイルはガンガン捨ててしまってくだせえ。どうせミスってもまた落とせるんで。

まあこの辞書を使う人は、ゆめ2っきの更新のたびにあのバカでかいzipファイルを落として解凍してる忍耐のある方だと思われるので…大丈夫でしょう！
